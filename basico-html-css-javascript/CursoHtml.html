Secçaõ de uma página

O elemento nav representa uma seção de uma página com links para outras páginas ou partes da página: uma seção com links de navegação.

O elemento ao lado representa uma seção de uma página que consiste em um conteúdo tangencialmente relacionado ao conteúdo ao redor do elemento ao lado e que pode ser considerado separado desse conteúdo. Essas seções são frequentemente representadas como barras laterais na tipografia impressa.

O elemento header representa o conteúdo introdutório para o conteúdo de seccionamento de seu ancestral mais próximo ou elemento raiz de seccionamento. Um cabeçalho normalmente contém um grupo de ajudas introdutórias ou de navegação. Quando o ancestral mais próximo do conteúdo do seccionamento ou do elemento raiz do seccionamento é o elemento body, ele se aplica a toda a página.

O elemento principal representa o conteúdo principal do corpo de um documento ou aplicativo. A área de conteúdo principal consiste em conteúdo diretamente relacionado ou que se estende ao tópico central de um documento ou à funcionalidade central de um aplicativo.

O elemento footer representa um rodapé para seu ancestral mais próximo seccionando o conteúdo ou seccionando o elemento raiz. Um rodapé geralmente contém informações sobre sua seção, como quem o escreveu, links para documentos relacionados, dados de direitos autorais e semelhantes.
